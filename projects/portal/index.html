A web-based interface for the SPECULOOS survey

The SPECULOOS survey generates approx. 70TB of raw images per night, thanks to its 8 robotic telescopes located around the globe. Once these data transferred, its automatic reduction pipeline produces daily light curves that need to be inspected and analyzed by a large consortium. To ease this collaborative process, Peter Pihlmann Pedersen (University of Cambridge, back-end) and I (front-end) developed a web-based interface, allowing our teams to visualize various data products, as well as reporting any signal of interest to the rest of the consortium (see features below).

The frontend of the web-based interface is built using VueJS, with a backend following the LAMP model. Visualizations are built using the plotly library.
